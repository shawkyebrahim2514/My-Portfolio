#### <Highlight variant="area">Quoting Website</Highlight>

- Utilized Node.js with Express.js, EJS engine, CSS, GraphQL with Apollor Explorer, JWT, MongoDB database, and MongoDB Atlas Search to create a full-stack website for exchanging quotations.
- Integrated JWT (JSON Web Tokens) enables secure user authentication and authorization, guaranteeing a simple and safe login process.
- Used the autocomplete functionality of MongoDB Atlas Search to provide quick quote searches.
- Designed user-friendly interfaces with form validation feedback for an improved UX.
