#### <Highlight variant="area">Muslim Website</Highlight>

- Created an Islamic website using ReactJS and Material UI components.
- Built a single resource for Muslims that included essential content such as the Holy Quran, the Hijri Calendar, and Adhkar.
- Utilized various technologies, including React Hook, memoization, React Router, React Context API with useReducer and Immer, React custom hooks, Lazy loading, and Separation of concerns
