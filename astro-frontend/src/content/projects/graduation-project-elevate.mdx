### 🎓 Graduation Project – ELEVATE

**An AI-powered e-commerce platform supporting 200+ Egyptian local fashion brands**

**Role:** Backend & AI Integration Developer  
**Team Size:** 6
**Tech Stack:** Node.js, Express.js, Firebase, Vercel, Algolia, Upstash, Google Vertex AI, Replicate, Swagger

ELEVATE is a mobile-first digital marketplace that connects consumers with Egyptian local clothing brands through a seamless shopping experience enhanced by artificial intelligence. Users can search for clothing using text or images, receive personalized recommendations, and virtually try on outfits using AI models—all in one centralized platform.

#### 💡 Key Features

- 🔍 **Visual Search**: Users upload a photo to find visually similar products using vector search and image embeddings.
- 🤖 **AI Try-On**: Virtual fitting powered by Replicate & Fal.AI, allowing customers to preview how clothes look on them.
- 🛍️ **Multi-Brand Shopping**: Unified access to 200+ Egyptian fashion brands with Arabic support and local payment options.
- 🧠 **Smart Recommendations**: Personalized suggestions using Algolia based on user behavior and browsing history.
- 📊 **Brand Owner Dashboard**: Analytics on sales, reviews, and subscriptions; full product management capabilities.

#### 🛠️ My Contributions

- Designed and built RESTful APIs using **Node.js** and **Express.js**
- Managed data flows with **Firebase Firestore**, Realtime DB, and Authentication
- Integrated **AI services** for visual search and try-on experiences
- Deployed scalable backend infrastructure on **Vercel**
- Created and maintained API documentation using **Swagger**
- Collaborated in an Agile team, contributing to system design, feature planning, and testing
